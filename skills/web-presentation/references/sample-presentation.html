why vanilla

왜 바닐라인가?

  • 01 의존성 0 — 빌드 도구, 번들러, CDN 불필요
  • 02 단일 HTML 파일로 즉시 실행 가능
  • 03 라이브러리 API 변경에 영향받지 않음
  • 04 완전한 코드 소유권과 커스터마이징 자유
features

핵심 기능

  • 01 키보드 / 터치 / 클릭 네비게이션
  • 02 GPU 가속 슬라이드 전환 애니메이션
  • 03 자동 순차 등장 효과 (auto-fragment)
  • 04 URL hash 동기화 + 프로그레스 바
  • 05 다크/라이트 테마 전환 (D키)

code

GPU 가속 전환

// Compositor 스레드에서 처리 — Layout/Paint 없음
.slide {
  transform: translate3d(100%, 0, 0); /* GPU 레이어 */
  opacity: 0;
  transition: transform 0.6s ease,
              opacity 0.6s ease;
}
.slide.active { transform: translate3d(0,0,0); opacity: 1; }

code

코드 예시

// transitionend 완료 감지 + 안전장치
const onEnd = (e) => {
  if (e.propertyName !== 'transform') return;
  clearTimeout(safetyTimer);
  cleanup(prevSlide);
};
to.addEventListener('transitionend', onEnd, { once: true });
const safetyTimer = setTimeout(cleanup, 700);
